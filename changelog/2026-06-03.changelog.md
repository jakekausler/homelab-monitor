## [STAGE-004-015] Explorer state persistence (last query/range/scroll across navigation)

**Summary:** The Logs Explorer remembers its last state — query, mode (plain/advanced), selected services, time range, and scroll position — when the user navigates away and returns to /logs. URL params take precedence over persisted state. localStorage with a 7-day TTL. Frontend-only (no backend/migration).

**Implementation (Option A — pure lib + Page seeding + scroll wiring colocated in Body):**
- `apps/ui/src/lib/explorerState.ts` (NEW) — pure core: ExplorerState/ExplorerSeed types, STORAGE_KEY 'homelab-monitor:logs-explorer-state', SCROLL_CONTAINER_ATTR constant, readState/writeState (SSR-guard + try/catch + per-field validation), saveExplorerState, loadExplorerState (7-day TTL → null when expired/corrupt), resolveInitialExplorerState (URL-precedence ALL-OR-NOTHING → ExplorerSeed incl. restoreScrollTarget), patchExplorerState (read-modify-write merge, two-writer-safe), rangeFromBounds/isPresetToken. cursor? field reserved (forward-compat; v1 does NOT resume pagination).
- `apps/ui/src/routes/logs/LogsExplorerPage.tsx` — seeds useState initializers once from resolveInitialExplorerState(urlParams, loadExplorerState()); useCallback-wrapped serializeCurrentExplorerState + write-back useEffect persisting query fields; passes restoreScrollTarget to Body. (013/014 saved-query/history behavior unchanged.)
- `apps/ui/src/routes/logs/LogsExplorerBody.tsx` — scroll capture + restore on the app's shared scroll container (AppShell's `<main data-app-scroll-container>`, resolved via querySelector — NOT the results `<pre>`, which only scrolls horizontally). Debounced 200ms scroll-save; restore-once via pendingRestoreRef + useLayoutEffect keyed on flatLines.length>0; restore gated by restoreScrollTarget (null when URL took precedence). Look-ahead comments for 018B (variable row heights) + 024 (live-tail suppress-restore).
- `apps/ui/src/components/AppShell.tsx` — marked the `<main>` scroll container with the shared SCROLL_CONTAINER_ATTR (data-app-scroll-container).

**Behavior:** restore query/services/range/mode + scroll position on navigate-back; URL params override persisted (all-or-nothing); 7-day TTL; persists across reload (localStorage); first-visit/expired → default.
