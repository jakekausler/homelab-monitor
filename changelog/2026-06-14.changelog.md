## [STAGE-005-028] Home Assistant drill-list widgets (frontend)

**Summary:** Added 5 frontend "drill-list" widgets to the Home Assistant panel, consuming the STAGE-005-027 VM-sourced detail endpoints. Each drill-list renders the actual rows behind the existing count tiles (which entities are unavailable, which batteries are low, etc.) so the user can drill from "962 unavailable" into the specific entities. Also fixed a pre-existing latent layout bug in the HA panel that prevented the tab bodies from scrolling.

**New files:**
- `apps/ui/src/routes/integrations/DrillList.tsx` — generic presentational shell `<DrillList<T>>` (loading / empty / error / 502 states, a `max-h-80` scroll container above a row threshold, a conditional "Showing {returned} of {total} — {ordering}" caption when total > returned, and a `keyExtractor` prop for stable list keys).
- `apps/ui/src/routes/integrations/HaEntitiesDrill.tsx` — container component owning `useHomeAssistantEntities` TanStack Query hook + independent loading/empty/error/502 state; renders unavailable entities (entity_id, domain, age).
- `apps/ui/src/routes/integrations/HaBatteriesDrill.tsx` — container component owning `useHomeAssistantBatteries` TanStack Query hook + independent loading/empty/error/502 state; renders low/critical batteries (entity_id, level% with red <10 / amber 10–<20 tint).
- `apps/ui/src/routes/integrations/HaUpdatesDrill.tsx` — container component owning `useHomeAssistantUpdates` TanStack Query hook + independent loading/empty/error/502 state; renders pending updates (title-or-entity_id fallback).
- `apps/ui/src/routes/integrations/HaConfigEntriesDrill.tsx` — container component owning `useHomeAssistantConfigEntries` TanStack Query hook + independent loading/empty/error/502 state; renders config-entry errors (domain, title, state badge).
- `apps/ui/src/routes/integrations/HaRepairsDrill.tsx` — container component owning `useHomeAssistantRepairs` TanStack Query hook + independent loading/empty/error/502 state; renders active repairs (domain, issue_id, severity badge).
- `apps/ui/src/routes/integrations/DrillList.test.tsx` — render tests (loading/empty/error/502 states, caption rendering, key extraction).
- `apps/ui/src/routes/integrations/HaEntitiesDrill.test.tsx` — hook-mocked render test (entity list, age formatting, 502 handling).
- `apps/ui/src/routes/integrations/HaBatteriesDrill.test.tsx` — hook-mocked render test (battery list, level tinting, 502 handling).
- `apps/ui/src/routes/integrations/HaUpdatesDrill.test.tsx` — hook-mocked render test (update list, title fallback, 502 handling).

- `apps/ui/src/routes/integrations/HaRepairsDrill.test.tsx` — hook-mocked render test (repair list, severity badge, 502 handling).
- `apps/ui/src/lib/relativeTime.test.ts` — tests for `formatAge(seconds)` helper (coarse single-unit "just now" / "Xm ago" / "Xh ago" / "Xd ago").

**Deferred work:** Fields the VM cannot carry (battery device name, update versions/release_url, repair summary, precise config-entry state) are deferred to STAGE-005-031 (live-HA enrichment) and STAGE-005-032 (config-entry state collector change). Frontend-only; consumes existing endpoints (no backend/OpenAPI change).

**Layout bug:** The HA tab bodies could not scroll when content exceeded the viewport (the `HomeAssistantLayout` Outlet wrapper is `overflow-hidden` and the tab bodies had no scroll region). Fixed by adding `h-full overflow-y-auto` to both tab-body root divs — establishes the scroll region without affecting the iframe/LogViewer (Logs/Metrics) tabs.

**Design decisions:** D-DRILL-EXPAND = always-visible lists below the count tiles (entities list gets a `max-h-80` scroll cap). D-DRILL-CAP = conditional "Showing {returned} of {total} — {ordering}" caption when total > returned. D-DRILL-ROW-SHAPE = one generic `<DrillList<T>>` shell + per-list `renderRow` (categorical state/severity → Badge; numeric battery level → tinted span; updates title falls back to entity_id). D-DRILL-DATA-WIRING = pure shell + thin per-list container owning its own hook + independent loading/empty/error/502 state.
