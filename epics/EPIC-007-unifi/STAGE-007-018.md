# STAGE-007-018: Nav restructure — remove Inventory page, relocate Crons under Integrations, add Integrations entries

## Status: Not Started

## Overview

The FIRST UI stage — restructure the sidebar so every subsequent EPIC-007 UI stage registers into the final
structure (build once, no throwaway wiring). Remove the top-level **Inventory** page (its only live content
is Crons; its other tabs are upcoming/unbuilt), **relocate Crons** under **Integrations ▸** (beside Docker
and Home Assistant), and add the Integrations entries the later stages fill in (Unifi, Network). This is a
**deviation from spec §9.1**, recorded as an intentional amendment.

**This relocates a CLOSED epic's UI surface (Crons, EPIC-002) — done here because we do not reopen closed
epics.** The Design phase MUST verify live exactly what the Inventory page contains today before removing it.

**Vertical slice:** after this stage the Inventory top-level page is gone; Crons is reachable under
Integrations with identical functionality; the sidebar shows the Integrations group with placeholders/entries
for Unifi + Network; no regression to the Crons view.

**Spec references:** spec §9.1 (nav — AMENDED here). EPIC-002 (the Crons UI being relocated; provenance
note added to EPIC-002). EPIC-001 (the UI shell + Inventory page being removed). EPIC-005 panel/sidebar
registration pattern (the target structure).
