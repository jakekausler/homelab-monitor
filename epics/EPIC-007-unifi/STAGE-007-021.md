# STAGE-007-021: Network page shell + WAN/DHCP/WiFi-experience/SSID widgets + DNS-posture indicator

## Status: Not Started

## Overview

The **Integrations → Network** page — network-as-a-concept (mostly Unifi-sourced). The page shell + widgets:
WAN/ISP status + speedtest history; **DNS-posture indicator** (clients handed `192.168.2.148` ✓ / drift —
the visual companion to `UnifiDnsSteeringDrift`); **DHCP pool usage** (authoritative lease count when the
opt-in lease collector is on, active-client approximation when off); WiFi experience/airtime; SSID client
distribution. The **Clients tab** + Client page land in the next stage (022).

**Vertical slice:** navigating to Integrations → Network renders the page with live WAN/DHCP/WiFi-experience/
SSID widgets + the DNS-posture indicator; the Clients tab placeholder is present (filled in 022).

**Spec references:** spec §9.2. EPIC-007 "UI structure" (surface 2) + "Verified deployment reality" (DNS
steering = DHCP handout). STAGE-007-019 (endpoints), STAGE-007-006/011 (WAN/DHCP data),
STAGE-007-012 (lease enrichment that sharpens the DHCP widget).
