# STAGE-008-029: Surveillance activity widgets + embedded `<LogViewer>` on the Synology page

## Status: Not Started

## Overview

Complete the UI: the Surveillance per-camera activity widgets (events/day, recording footprint,
recording-storage) and the embedded `<LogViewer>` (scoped `service:"synology-*"`) on the Synology page (the
EPIC-004 embedding contract). The LogViewer gives operator-facing line-level forensic search over the DSM
syslog stream (logins, SMART events, package events). FRONTEND — Desktop + Mobile sign-off.

**Vertical slice:** after this stage the Surveillance page shows per-camera activity + recording-storage, and
the Synology page embeds a working `<LogViewer>` filtered to `service:"synology-*"`.

**Spec references:** §9.2. EPIC-004 STAGE-004-003 (`<LogViewer>` embedding contract). EPIC-005 STAGE-005-025 +
EPIC-007 STAGE-007-023 (embedded LogViewer — the precedent). EPIC-008 STAGE-008-016/020 (event data + syslog
stream).
