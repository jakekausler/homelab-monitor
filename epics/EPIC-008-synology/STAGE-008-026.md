# STAGE-008-026: Synology widgets — storage / disks-SMART / RAID / system-UPS / SSH-probe

## Status: Not Started

## Overview

Fill the Synology page with the hardware/storage widgets (consuming the 008-024 detail endpoints): storage/
volume usage, the per-disk SMART grid (8 disks: model/temp/`smart_status`/`remain_life` + the full
per-attribute SMART from the SSH probe on drill-down), RAID/pool status, system/temps, UPS, and the
**SSH-probe state** card (last success, host-key status, duration — the EPIC-017 render mandate). FRONTEND —
Desktop + Mobile sign-off.

**Vertical slice:** after this stage the Synology page shows live storage/disk/SMART/RAID/system/UPS data +
the SSH-probe-state card.

**Spec references:** §9.2. EPIC-005 STAGE-005-023 + EPIC-007 STAGE-007-020 (widgets consuming detail data).
EPIC-017 UI mandate (consumer pages render `homelab_ssh_*` probe state). EPIC-008 Amendment 2/7.

## Concrete deliverables

### 1. Storage / volume + disk-SMART + RAID/pool widgets

Volume usage widget; per-disk SMART grid (status/temp/remain-life) with a per-attribute drill-down (the
`homelab_synology_smart_attr` table); RAID/pool status (degraded/rebuild progress/scrub).

### 2. System / temps / UPS + SSH-probe-state widgets

System info + temps card; UPS card (charge/runtime/status); SSH-probe-state card (last success, host-key
status, duration).

### 3. Tests + viewport sign-off

Component/vitest tests; Refinement Desktop + Mobile sign-off on the prod UI.
