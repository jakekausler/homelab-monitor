# STAGE-006-011: Version/update collector

## Status: Not Started

## Overview

Poll `/api/info/version` and emit update-available flags per component (core/web/ftl/docker) by diffing
each component's local vs remote version. Feeds Grafana row 7 + the panel version widget + the
`PiholeUpdateAvailable` info alert. The live host currently shows Web + Docker updates available — good
test data.

**Vertical slice:** `homelab_pihole_update_available{component}` (1/0) in VM; Web + Docker show `1` on the
live host.

**Spec references:** EPIC-006 "Metric families" (Version/update). Alert: `PiholeUpdateAvailable` (info).

## Concrete deliverables

### 1. Collector

Poll `/api/info/version` (~1h). For each component (core/web/ftl/docker), compare `local.version` vs
`remote.version`; emit `homelab_pihole_update_available{component}` = 1 when they differ, 0 when equal.
Optionally also emit the current version strings as an info-gauge (`homelab_pihole_version_info{component,
version}` value 1) for the panel/dashboard, bounded cardinality (4 components).
