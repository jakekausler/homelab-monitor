# STAGE-006-013: Unbound stats collector

## Status: Complete

## Overview

Emit unbound recursive-resolver metrics using the 006-003 access layer: cache hit ratio, recursion-time
quantiles (avg/median + histogram p50/p95/p99), request-list depth, and — now that extended-statistics is
enabled — DNSSEC validation outcomes (secure/bogus), answer RCODEs (NXDOMAIN/SERVFAIL), and per-RR-type
counts. Feeds Grafana row 6 and the four Unbound alerts. Gracefully degrades to the default key set if
extended-stats is ever off.

**Vertical slice:** `homelab_unbound_*` metrics in VM — cache hit ratio, recursion p50/95/99,
`_answer_secure_total`/`_answer_bogus_total`, `_answer_rcode{rcode}`, `_query_type{type}`,
`_extended_stats_enabled=1` on the live host.

**Spec references:** EPIC-006 "Metric families" (Unbound), "Verified deployment reality" (extended-stats
enabled). Consumes 006-003. Alerts: `UnboundDnssecBogusSpike`, `UnboundServfailSpike`, `UnboundRecursionSlow`,
`UnboundCacheHitLow`.

## Cross-cutting standards

- Consume 006-003 (exec + parse + extended detection). Emit `homelab_pihole_unbound_extended_stats_enabled`.
- Graceful degrade: extended-derived metrics ONLY when keys present; default metrics always.
- Counters are real monotonic unbound counters → safe for `rate()` (unlike Pi-hole's 24h-rolling summary).

## Concrete deliverables

### 1. Collector

~60s. Always emit (default set): `homelab_unbound_queries_total`, `_cache_hits_total`, `_cache_misses_total`,
`_cache_hit_ratio` (derived), `_prefetch_total`, `_recursion_time_seconds{quantile="avg"|"median"}`,
`_requestlist_current`, `_requestlist_exceeded_total`. When extended present, ALSO emit:
`_recursion_time_seconds{quantile="0.5"|"0.95"|"0.99"}` (from `histogram.*`), `_answer_rcode{rcode}`
(NOERROR/NXDOMAIN/SERVFAIL/REFUSED/…), `_answer_secure_total`, `_answer_bogus_total`, `_query_type{type}`.

### 2. Histogram → quantiles

Derive p50/p95/p99 from the unbound `histogram.*` buckets (Design picks the bucket-interpolation approach;
note unbound's histogram is fixed log-scale buckets).

## Design Phase

### Design Notes

**Approach selected (coordinator selected own recommendation; full Opus brainstormer consulted — docker-exec + histogram math are novel).** Unbound stats collector consuming the 006-003 access layer via docker-exec.

**Recon-first (live fixtures + codebase ground truth, captured in Design):**
- Access layer: `fetch_unbound_stats(*, exec_backend: ExecCapture, container: str, timeout_seconds=5.0) -> UnboundStats | UnboundError` (`kernel/pihole/unbound_control.py`). `UnboundStats(raw: dict[str,float], extended_enabled: bool, raw_line_count: int)` — NO timing field. `UnboundError(reason: Literal[container_unreachable|socket_error|control_error|empty_output|parse_error], message)`. `ExecCapture` Protocol satisfied by `DockerSocketClient`.
- unbound IS a docker container (`pihole-unbound`); access = `docker exec unbound-control stats_noreset`. NOT a LAN service (corrected a recon mis-inference).
- **`PiholeUnboundConfig(container="pihole-unbound")` + `load_pihole_unbound_config()` EXIST** (`config.py:666-687`, PUBLIC, env `HOMELAB_MONITOR_PIHOLE_UNBOUND_CONTAINER`), declared by 003 but not yet wired. The collector uses them (do NOT hardcode the container name).
- Real extended fixture: BOTH `thread0.*` and `total.*` prefixes (single-thread exporting aggregates). 41 log-scale histogram buckets `histogram.{lo_sec}.to.{hi_sec}={count}` (microsecond boundaries to 0.524288s, then seconds). 10 `num.query.type.*`, 7 `num.answer.rcode.*` (incl. lowercase `nodata`), `num.answer.secure`/`num.answer.bogus`/`num.rrset.bogus`. recursion.time.avg/median are in SECONDS already. Default fixture = default set only (no histogram/query.type/rcode/secure).
- Docker-socket injection: `CollectorContext` does NOT carry the socket; it's injected post-construction in `lifespan.py` via `isinstance(c, ...)` blocks setting `c._socket_client = getattr(app.state, "docker_socket_client", None)` (the DockerSocketCollector block, gated on `docker_config.enabled`, creates+stores it). Pihole collectors register UNCONDITIONALLY (no docker gate) → the unbound collector's `if self._socket_client is None` run-guard is the correct degrade path when docker is off.

**Locked decisions (full Opus brainstormer; coordinator accepted all + folded in 2 verified corrections — PiholeUnboundConfig exists; api_took is collector-self-measured):**
1. **Histogram → quantiles:** linear interpolation within the containing bucket (Prometheus `histogram_quantile`-style), in a PURE module-level helper `_histogram_quantiles(buckets, total) -> dict`. Parse `histogram.{lo}.to.{hi}` bounds (6.6 digit format → seconds float). Build cumulative; per p∈{0.5,0.95,0.99}: rank=p×total, interpolate `lo + (hi-lo)×(rank-cum_before)/count`, CLAMP to `hi` (open tail). **Zero-total / empty-histogram → SKIP the quantile emits** (never emit 0/NaN); avg/median still emit (different source). Malformed bucket key → skip that bucket (coverage branch).
2. **Source keys:** `total.*` preferred, `thread0.*` fallback, via one shared `_read(raw, suffix)` helper. Histogram keys are global (no thread prefix). Skip emit when a key is absent (never emit 0).
3. **Units:** seconds, no conversion. avg/median emit UNCONDITIONALLY as `{quantile="avg"|"median"}`; histogram p50/p95/p99 as `{quantile="0.5"|"0.95"|"0.99"}` on the SAME metric `homelab_unbound_recursion_time_seconds` (string + numeric-string quantile labels coexist fine — distinct series).
4. **Naming (card verbatim — deliberate documented decision, zero downstream contract change for 016/026):** default metrics `homelab_unbound_*`; the extended-stats flag keeps the card's `homelab_pihole_unbound_extended_stats_enabled` (pihole_ prefix = bundle-scoped indicator, parallel to `homelab_pihole_api_took_seconds`). Did NOT "fix" the prefix to avoid breaking the 016/026 contract.
5. **Docker-disabled handling:** `__init__(self, *, socket_client: DockerSocketClient | None = None, unbound_config: PiholeUnboundConfig | None = None)`; guard `if self._socket_client is None: return CollectorResult(ok=False, errors=["client_unconfigured"], 0 emits)` at top of run() (mirrors DockerSocketCollector). New lifespan `isinstance(c, UnboundStatsCollector)` block (after the DockerSocketCollector block so `app.state.docker_socket_client` is set first) reusing the shared socket + `load_pihole_unbound_config()`.
6. **Location/registration:** new file `plugins/collectors/integrations/pihole/unbound_stats.py`, class `UnboundStatsCollector`, `name="unbound_stats"`, `interval=60s`, `timeout=15s`, `concurrency_group="pihole"`. Append to `_PIHOLE_COLLECTORS` (one-line) — lifespan isinstance-injection still applies (loop iterates ALL loaded collectors). Uses `self._socket_client`/`self._cfg`, NOT `ctx.pihole`; uses `ctx.vm.write_gauge`/`ctx.log`.

**api_took (collector self-measured):** the helper provides NO timing (`UnboundStats` has no took field), so the collector measures wall-clock (`time.monotonic()` around `fetch_unbound_stats`) and emits `homelab_pihole_api_took_seconds{endpoint="unbound/stats_noreset"}` — a collector-observes-itself signal (project mandates self-observation), keeping the bundle's universal api_took contract.

**Final metric-name list (pin in a `test_metric_name_constants_match_contract` literal test):**
Always emitted (default): `homelab_unbound_queries_total` (←total.num.queries), `homelab_unbound_cache_hits_total` (←total.num.cachehits), `homelab_unbound_cache_misses_total` (←total.num.cachemiss), `homelab_unbound_cache_hit_ratio` (derived hits/(hits+misses); SKIP on zero denom), `homelab_unbound_prefetch_total` (←total.num.prefetch), `homelab_unbound_recursion_time_seconds{quantile="avg"|"median"}` (←total.recursion.time.avg/median), `homelab_unbound_requestlist_current` (←total.requestlist.current.all), `homelab_unbound_requestlist_exceeded_total` (←total.requestlist.exceeded), `homelab_pihole_unbound_extended_stats_enabled` (1/0 from UnboundStats.extended_enabled), `homelab_pihole_api_took_seconds{endpoint="unbound/stats_noreset"}` (collector wall-clock).
Extended-only: `homelab_unbound_recursion_time_seconds{quantile="0.5"|"0.95"|"0.99"}`, `homelab_unbound_query_type{type}` (dynamic from `num.query.type.*`), `homelab_unbound_answer_rcode{rcode}` (dynamic from `num.answer.rcode.*`, pass label verbatim incl. lowercase `nodata`), `homelab_unbound_answer_secure_total` (←num.answer.secure), `homelab_unbound_answer_bogus_total` (←num.answer.bogus).

**Intentional omission (NOT a deferral — extra data the card never requested):** `num.rrset.bogus` is present in the fixture but NOT in the card's deliverable set; left unemitted (mirrors upstreams.py "not emitted here" precedent). This is an idea-note, not a tracked drop. **No deferrals.** Backend-only; no migration, no API/frontend change.

**Files:** CREATE `plugins/collectors/integrations/pihole/unbound_stats.py` + `tests/test_unbound_stats_collector.py`; MODIFY `plugins/collectors/integrations/pihole/__init__.py` (import + `_PIHOLE_COLLECTORS` append) + `kernel/api/lifespan.py` (isinstance injection block).
