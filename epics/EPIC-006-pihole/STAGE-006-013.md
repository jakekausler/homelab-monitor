# STAGE-006-013: Unbound stats collector

## Status: Not Started

## Overview

Emit unbound recursive-resolver metrics using the 006-003 access layer: cache hit ratio, recursion-time
quantiles (avg/median + histogram p50/p95/p99), request-list depth, and — now that extended-statistics is
enabled — DNSSEC validation outcomes (secure/bogus), answer RCODEs (NXDOMAIN/SERVFAIL), and per-RR-type
counts. Feeds Grafana row 6 and the four Unbound alerts. Gracefully degrades to the default key set if
extended-stats is ever off.

**Vertical slice:** `homelab_unbound_*` metrics in VM — cache hit ratio, recursion p50/95/99,
`_answer_secure_total`/`_answer_bogus_total`, `_answer_rcode{rcode}`, `_query_type{type}`,
`_extended_stats_enabled=1` on the live host.

**Spec references:** EPIC-006 "Metric families" (Unbound), "Verified deployment reality" (extended-stats
enabled). Consumes 006-003. Alerts: `UnboundDnssecBogusSpike`, `UnboundServfailSpike`, `UnboundRecursionSlow`,
`UnboundCacheHitLow`.

## Cross-cutting standards

- Consume 006-003 (exec + parse + extended detection). Emit `homelab_pihole_unbound_extended_stats_enabled`.
- Graceful degrade: extended-derived metrics ONLY when keys present; default metrics always.
- Counters are real monotonic unbound counters → safe for `rate()` (unlike Pi-hole's 24h-rolling summary).

## Concrete deliverables

### 1. Collector

~60s. Always emit (default set): `homelab_unbound_queries_total`, `_cache_hits_total`, `_cache_misses_total`,
`_cache_hit_ratio` (derived), `_prefetch_total`, `_recursion_time_seconds{quantile="avg"|"median"}`,
`_requestlist_current`, `_requestlist_exceeded_total`. When extended present, ALSO emit:
`_recursion_time_seconds{quantile="0.5"|"0.95"|"0.99"}` (from `histogram.*`), `_answer_rcode{rcode}`
(NOERROR/NXDOMAIN/SERVFAIL/REFUSED/…), `_answer_secure_total`, `_answer_bogus_total`, `_query_type{type}`.

### 2. Histogram → quantiles

Derive p50/p95/p99 from the unbound `histogram.*` buckets (Design picks the bucket-interpolation approach;
note unbound's histogram is fixed log-scale buckets).
