# STAGE-006-007: Gravity + per-adlist health collector

## Status: Not Started

## Overview

Monitor Pi-hole's blocklist engine ("gravity"): the aggregate domain count + last-update age, AND
**per-adlist health** (each adlist's enabled state, domain contribution, and last-download status). The
live host currently has 2 failing adlists (inaccessible during the last gravity run) тАФ this collector +
the `PiholeAdlistFailing` alert make that specific and actionable. Feeds Grafana row 4 and the gravity/
adlist alerts.

**Vertical slice:** `homelab_pihole_gravity_domains` + `_gravity_last_update_age_seconds` +
`_adlist_domains{list}` / `_adlist_enabled{list}` / `_adlist_status{list}` appear in VM; the 2 failing
adlists show a failed status by name.

**Spec references:** EPIC-006 "Metric families" (Gravity + per-adlist). Q5 decision = option B (essential +
per-adlist). Alerts: `PiholeGravityStale` (TZ-guarded), `PiholeGravityDomainsDropped`, `PiholeAdlistFailing`.

## Cross-cutting standards

- Reuse client session; failure budget; self-metrics.
- `gravity_last_update_age` computed as `now - last_update` тАФ Design must TZ-guard (a stale-looking
  timestamp can be a timezone artifact, not a real failure; research-flagged) so `PiholeGravityStale`
  doesn't false-fire.
- Per-adlist cardinality: bounded by the number of adlists (small) тАФ label by list id/address; cap if a
  user has a huge adlist count (configurable).

## Concrete deliverables

### 1. Gravity aggregate

From `/api/info/ftl` (gravity domain count + last_update) тАФ emit `homelab_pihole_gravity_domains` and
`homelab_pihole_gravity_last_update_age_seconds` (TZ-correct).

### 2. Per-adlist health

From `/api/lists` (~5min) тАФ emit per-adlist `homelab_pihole_adlist_domains{list}`,
`_adlist_enabled{list}` (1/0), `_adlist_status{list}` (enum: ok / failed / тАж; map the v6 list-status field).
The `list` label = the adlist id or address (Design picks the stable key; address is human-meaningful).
Surface the failing-adlist case (the live 2) as a `failed` status.
